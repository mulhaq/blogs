HOW TO BUILD & RUN
================================================================================

1. Prerequisites:
   - Java 21+ installed
   - Maven 3.8+ installed
   - Internet connection (for APIs)

2. Build:
   cd /workspace/java-mcp-weather-news
   mvn clean package

3. Run:
   mvn spring-boot:run

4. Access:
   Open http://localhost:8080 in your browser

5. Test:
   Try: "What's the weather in Ashburn, VA?"
        "Should I go out today in New York?"
        "Tell me the latest news"

For detailed instructions, see BUILD_AND_RUN.md
