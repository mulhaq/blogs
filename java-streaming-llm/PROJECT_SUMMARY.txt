================================================================================
HOW TO RUN
================================================================================

1. Set Groq API Key:
   export GROQ_API_KEY="your-groq-api-key-here"

2. Build and run:
   cd /workspace/java-streaming-llm
   mvn clean spring-boot:run

3. Open browser:
   http://localhost:8080

4. Type a message and press Enter or click Send
   → Tokens will stream in real-time from Groq's LLM

ASSUMPTIONS & NOTES
================================================================================

1. GROQ_API_KEY environment variable will be set before running
   - If not set, Spring Boot will fail at startup with clear error message
   
2. Client browser supports ReadableStream and fetch API
   - All modern browsers (Chrome 43+, Firefox 65+, Safari 13+, Edge 79+)
   
3. Port 8080 is available on localhost
   - Can be changed in application.properties if needed
   
4. Using llama-3.3-70b-versatile model from Groq
   - Fastest open-source LLM at inference time
   - Can be swapped for other Groq models by editing application.properties

5. SSE timeout set to 30 seconds
   - Appropriate for most queries
   - Can be increased if needed (see README troubleshooting)

6. Virtual thread executor (Java 21 feature)
   - Creates lightweight threads for each concurrent stream
   - Far more efficient than traditional thread pool
   - Allows hundreds/thousands of concurrent connections
